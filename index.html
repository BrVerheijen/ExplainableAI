Wat is

Een van de populairste onderzoeksonderwerpen in het brede veld van Artificial Intelligence is Reinforcement Learning. Reinforcement Learning (RL) is een vorm van machine learning dat een agent in staat stelt om te leren in een interactieve omgeving. Dit gebeurt door middel van feedback die de agent ontvangt op basis van de acties die de agent in de omgeving neemt. Reinforcement Learning maakt gebruik van rewards en punishments voor goed en slecht gedrag. De agent bevindt zich continu in een bepaalde situatie oftewel state. Binnen de environment neemt de agent een bepaalde actie dat resulteert in een nieuwe state en feedback in de vorm van rewards. Het doel van Reinforcement Learning is om een geschikt model van acties te vinden dat de totale cumulatieve reward van de agent maximaliseert.

Experimenten

Timesteps

Het programma is getraind op verschillende aantallen timesteps, hierdoor is te zien hoe het programma zichzelf ontwikkeld.

Courses

Het programma heeft op een aantal courses getraind, hierdoor is te zien hoe het programma zich ontwikkeld op verschillende courses.

Characters

Het programma is getraind met alle verschillende character archetypes, hier is te zien hoe het programma zich ontwikkeld op ieder archetype.

Grand Prix

Het programma is getraind op grand prix modus om te zien of het programma ook kan trainen wanneer er meerdere spelers zijn.

Reward Functions

Het programma is getraind op verschillende reward functions om er achter te komen welke variablen het programma het meest sturen.

Resultaat

Eindresultaat

Het eindresultaat van ons project is een simpel programma dat in staat is om Super Mario Kart (Snes) te spelen. Dit wordt gedaan door het programma op een bepaalde course te trainen, hier ontvangt het programma tijdens het trainen steeds rewards en slaat het het model op wat tot nu toe het beste getraind is. Hierdoor wordt het programma steeds iets beter in het leren van de course en kan hij deze uiteindelijk constant finishen.


Wij hebben voor de ontwikkeling van dit project gebruik gemaakt van OpenAI Gym Retro (retro) en Stablebaselines3. Hier is retro gebruikt voor het opzetten van de omgeving, waarin de agent begeeft en acties uitvoerd. Stablebaselines3 is gebruikt om een leeralgoritme toe te voegen en om inzichten te krijgen in hoe de agent zich ontwikkeld. Daarnaast is gebruik gemaakt van een bestaand project wat hetzelfde doel had als ons project.
